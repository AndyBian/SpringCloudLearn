有点问题一直没有提醒大家，EurekaServer等几个项目用的host是本地写好的，在本系列文章的第一文中有写，但是还是有人不知道，
这里为大家提个醒
在hosts文件添加
```
127.0.0.1 peer1
127.0.0.1 peer2
127.0.0.1 discovery
```
